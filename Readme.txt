****How to run the 'Bookish - Book Library System' Project****

1. Download and Unzip file on your local system copy Bookishphp
2. Put Bookishphp folder inside root directory

Database Configuration

Open phpmyadmin
Create Database library
Import database library.sql (available inside zip package)
